人工验收步骤

  • 打开 http://localhost:3000/#providers，确认 OpenAI Codex OAuth 显示 0/1。
  • 点击 OpenAI Codex OAuth 分组，确认弹窗里总账户数为 1，并能看到 1 条账号行。
  • 打开用量查询，选择或刷新 Codex，确认出现 Daily Tokens、Weekly Tokens、Total Tokens 三个 token 明细项。
  • 该账号若在提供商池仍显示异常或不健康，说明内容生成链路曾触发过上游 OAuth 失效；这不影响本次用量查询修复的验收。
  • 重新授权 Codex 账号后，再用模型测试或 /v1/chat/completions 做内容生成验收。
